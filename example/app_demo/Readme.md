# test

A TSSPA project.

You can also use cli to create view.

## Dev
1. install dependencies
```bash
yarn install
```
2. genrate dll filename
```bash
yarn dll
```

3. start webpack dev server
```bash
yarn dev
```

## Prod
1. generate build files
```bash
yarn build
```
2. start a express prod server
```bash
yarn start:prod
```

## CLI
you can use CLI to create view.
```
yarn cli -v
```
